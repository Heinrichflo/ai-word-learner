# AI 背单词小程序

> 一个 AI 做的背单词产品，7天从痛点到 MVP

## 🚀 5分钟快速开始

### 方法一：网页版（最简单）

1. 下载项目：点击 Code → Download ZIP
2. 解压
3. 双击 `index.html` 用浏览器打开

### 方法二：Chrome 插件

1. 打开 `chrome://extensions/`
2. 开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `chrome-extension` 文件夹

## 功能

- 🤖 AI 出题
- 🗣️ 发音纠正
- 📅 艾宾浩斯记忆
- 📊 学习统计
- 🔌 Chrome 插件
